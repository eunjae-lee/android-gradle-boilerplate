Detailed Description

Provides all available methods for the FlurryAdListener interface.

Set of methods that allows a developer to manage and take actions within different phases of App ad display.

Note
For additional information on how to use Flurry's Ads SDK to attract high-quality users and monetize your user base see Support Center - AppCircle.

boolean shouldDisplayAd ( String  adSpace,
FlurryAdType  type 
)

Invoked when an ad will display on the specified adSpace.

Since
3.0.0

This method informs the app that an ad will display to the user. You can use this to pause app states. You can also decide not to show this ad by simply returning false.

See Also
FlurryAgent::showAdForSpace for details on the method that will invoke this delegate.
Parameters
adSpaceThe placement of an ad in your app, where placement may be splash screen for SPLASH_AD.
typeThe type of ad to be displayed. See FlurryAdType for allowable values.
Returns
true if the ad should be displayed, false if not
